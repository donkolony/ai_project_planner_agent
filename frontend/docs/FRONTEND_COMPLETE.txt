🔌 API INTEGRATION

Base URL: http://localhost:8000

Endpoints Used:
  POST /plan/          - Generate new plan
  GET  /health         - Health check
  GET  /plan/{id}      - Get specific plan
  GET  /plan/          - List all plans

Request Format:
  {
    "project_name": "string",
    "description": "string",
    "tech_stack": ["string", ...]
  }

Response Format:
  {
    "summary": "string",
    "phases": [
      {
        "name": "string",
        "tasks": ["string", ...]
      }
    ]
  }

📞 SUPPORT & TROUBLESHOOTING

Common Issues & Solutions:

1. "Cannot find module 'react'"
   → npm install

2. "Port 5173 already in use"
   → npm run dev -- --port 5174

3. "Backend connection failed"
   → Ensure backend running at http://localhost:8000
   → Check CORS configuration
   → Verify API endpoints

4. "Form submission doesn't work"
   → Check browser console
   → Verify backend endpoint: POST /plan/
   → Check Network tab for request/response

5. "Styling looks wrong"
   → Clear browser cache (Ctrl+Shift+Delete)
   → Restart dev server
   → Check tailwind.config.js
