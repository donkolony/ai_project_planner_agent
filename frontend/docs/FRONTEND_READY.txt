🎬 USER WORKFLOW

1. User visits application
   ↓
2. Fills in project details
   - Name
   - Description
   - Tech stack
   ↓
3. Clicks "Generate Plan"
   ↓
4. Sees loading spinner
   ↓
5. API generates plan (via backend)
   ↓
6. Navigates to results page
   ↓
7. Views plan components
   - Architecture
   - Timeline roadmap
   - Task board
   - GitHub issues
   ↓
8. Copies issues to GitHub
   ↓
9. Can create new plans
